## d3-simple-charts

React wrapper on [d3 version 4.x](https://github.com/d3/d3/wiki) charts realization: `<BarChart />`, `<DonutChart />`, `<PieChart />`.
Package based on [CRA boilerplate](https://github.com/facebook/create-react-app).

### Steps to run

- fork or clone repo
- `npm install`
- `npm start`
